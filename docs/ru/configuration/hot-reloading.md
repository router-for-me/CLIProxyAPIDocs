# Горячая перезагрузка

Сервер отслеживает изменения в файле конфигурации и `auth-dir`, автоматически перезагружая настройки и клиенты.

Вы можете добавлять или удалять JSON-файлы токенов `Gemini CLI` / `Codex` / `Claude Code` / `Qwen Code` / `iFlow` во время работы сервера; перезагрузка не требуется.
